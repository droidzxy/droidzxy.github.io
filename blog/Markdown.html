# Markdown语法

# 一二级标题(三个以上在下方)：

使用###和---标记一级和二级标题

# 六级标题(个数即等级)：

使用#标记标题，分为1-6级

# 段落：

空行加回车代表一个新的段落

# 字体：

*斜体*，_斜体_， **粗体**， __粗体__， ***粗斜体***， ___粗斜体___，~~删除线~~

# 分割线(单独占一行)：

***， ****， ---， ___， ------， _____ 代表分割线

# 脚注：

文本[^脚注标记]， [^脚注标记]:这是对应的详细脚注

# 无序列表(字符后有空格)：

\ * 第一项， * 第二项，+ 第一项，+ 第二项， - 第一项， - 第二项

# 有序列表(.后有空格)：

\ 1. 第一项， 2. 第二项，3. 第三项

# 列表嵌套：

子列表前加4个空格

# 区块：

\>区块内容， >>嵌套区块

# 代码文字(反引号)：

printf()

# 代码区块(四空格或三反引号)：

echo ‘Hello’，```javascript ```

# 链接：

[链接名称](链接地址)，<链接地址>

# 图片：

![alt属性文本](图片地址)

# 表格：

|左对齐|居中对齐|右对齐| n |:---|:---:|---:|
