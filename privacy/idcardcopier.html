# 隐私声明

本隐私声明由开发者『周希元』向您提供，请您以严谨的态度，逐字逐句认真阅读，并确保知晓其中所有细节；如您拒不阅读或一跳而过，我们拿你也没有任何办法；但为了保证您的个人利益受到应有的法律保护，还是请您逐字逐句认真阅读，即使您觉得没有任何必要。

# 一、引言

我们非常重视用户的隐私。您在使用我们的服务时，我们可能会征求您打开相关的权限。我们希望通过本《隐私声明》向您说明，在使用我们的服务时，我们如何使用、储存这些信息，以及我们为您提供的访问、更新、控制和保护这些信息的方式。本《隐私声明》与您所使用的服务息息相关，希望您仔细阅读，在需要时，按照本《隐私声明》的指引，作出您认为适当的选择。本《隐私声明》中涉及的相关技术词汇，我们尽量以简明扼要的表述，并提供进一步说明的链接，以便您的理解。

您使用或继续使用我们的服务，即意味着同意我们按照本《隐私声明》使用您的设备的相关功能。

如对本《隐私声明》或相关事宜有任何问题，请通过droidzxy@163.com与我们联系。

# 二、我们采集的信息

我们提供服务时，可能会采集您有关的信息。如果您不提供相关信息，可能无法享受我们提供的某些服务，或者无法达到相关服务拟达到的效果。在拍照时，我们需要对您提供的证件进行照片采集，采集的照片信息存储在应用内部私有控件，并不会允许其他应用访问，也不会上传到任何服务器，而且照片信息不会无限占用你的控件，它会覆盖存储。

# 三、您提供的信息

您在使用我们的服务时，向我们提供的相关个人信息，例如证件图片等；

# 四、需要访问的您设备功能

访问相机：允许应用拍摄照片

操作存储卡上的内容：允许应用读取和修改存储卡上的照片和文件

# 五、我们如何使用信息

我们可能将在向您提供服务的过程之中所收集的信息进行如下处理：

  1. 存储到缓存； 在我们采集图片时，会将图片暂存在应用缓存中，外界不可见，其他应用也不可见；

  2. 对图片裁剪： 我们对图片裁剪时，依旧是在缓存中进行，结果还是保存在缓存中，

  3. 对图片处理： 我们对图片调整时，依旧是在缓存中进行，结果还是保存在缓存中，

  4. 复印件的保存： 生成成功后，文件为JPEG格式图片, 保存的路径在「我的手机/Pictures/idcard/」目录。

# 六、分享的信息

我们不会做任何分享的操作，用户所有操作与结果都是在本机执行，不需要联网。

# 七、用户守则

  1. 遵守所有的法律法规；

  2. 遵守法院命令或其他法律程序的规定；

  3. 遵守相关政府机关的要求；

# 八、信息安全

软件使用过程中不会备份保留您的个人信息，操作一旦完成，文件只生成保存在手机相册中，软件再卸载或重新安装不会有任何影响。

# 九、未成年人使用我们的服务

我们鼓励父母或监护人指导未满十八岁的未成年人使用我们的服务。我们建议未成年人鼓励他们的父母或监护人阅读本《隐私声明》，并建议未成年人在提交的个人信息之前寻求父母或监护人的同意和指导。

# 十、我们如何保护您的个人信息

  1. 我们会采取一切合理可行的措施，保护您的个人信息。例如，不访问网络，只在内存中操作文件；

  2. 手机并非绝对安全的工具，而且而且大部分手机都会联网，我们建议您使用完毕，即刻删除相关文件，避免病毒或其他软件非法访问您的资源；

  3. 如果您的隐私遭到泄漏，请求助于当地公安机关；

# 十一、您应如何管理您的个人信息

  1. 对于生成的文件，不随意发给其他人；

  2. 不随意存储保存复印件到网络上；

  3. 用完最好删除，下次再用时，重新生成；

# 十二、如何联系作者
