# 一个普通的80后程序员

这个网站主要介绍作者做的一些小软件，方便自己和有需要的朋友下载，因为随着国家政策和应用商店规则的变化，很多软件长时间不更新，就会从应用商店被强制下架。

# 吐槽：

毕业之初想有一个自己的网站，现在终于梦想成真了。 曾经一名普通的本科毕业生，现在依旧普通，我终究没有成为例外，尽管和大部分人一样，最初也有着自己的梦想和追求。 生活和工作让本来的我磨平了棱角，以前的年轻气盛，趾高气昂退化成了随波逐流、与世无争。 很少有人能生活在理想的状态中，但我们都在为此在努力，尽管结果不尽如人意，但我们依然不愿放弃。 不再年轻，却不年老，珍惜每一天，快乐每一天。 我仿佛听见十年后的自己对现在的我大声呐喊：“加油！别泄气！你比我年轻！”

如果无法改变生活，那就改变自己吧。尽力而为，随遇而安！
